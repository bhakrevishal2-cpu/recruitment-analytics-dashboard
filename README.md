Maine yeh project isliye banaya hai kyunki aaj ke time mein companies ke liye hiring aur recruitment ka data track karna kaafi messy ho jata hai. Yeh website ek simple aur clean solution hai jahan par HR aur companies apne poore hiring process ka data ek hi jagah par aasaani se dekh sakti hain. Basically, yeh recruitment ka control center hai! 🎯

🛠 Yeh website kya kaam karti hai?
Yeh ek powerful analytics tool hai. Jab koi company naye logo ko hire karti hai, toh unhe bahut sara data manage karna padta hai. Yeh dashboard us boring data ko aasaan graphs aur numbers mein convert karta hai taki decision lena easy ho jaye.

📊 Track Candidates: Kitne logo ne apply kiya, kitne interview round mein pahuche, kitne select hue, aur kitne reject hue, yeh sab ek click par mil jayega.

📈 Analytics aur Insights: Hiring trends ko samajhne ke liye visual data aur charts hain isme.

💻 Easy Management: Baar baar alag alag spreadsheet kholne ka tension nahi, ek hi screen par saari important details mil jati hain.

💻 Tech Stack (Maine kya use kiya hai)
Code ko clean rakhne ke liye maine frontend aur backend dono ko alag folders mein rakha hai.

Backend: Python aur FastAPI 🐍

Frontend: React / HTML CSS JS ⚛️

Deployment: Render par live kiya hai 🌐

🚀 Isko apne PC par kaise chalaye?
Agar tumhe mera code apne system par run karna hai, toh bas ye steps follow karo:

Backend setup:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Frontend setup:
cd frontend
npm install
npm start
